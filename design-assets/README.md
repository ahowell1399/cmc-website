# Design Assets — drop zone

Put raw source files here. These are **originals for reference/archival** — they are
NOT deployed as-is; the build optimizes copies of them into the site. Drop anything
you have; don't worry about resizing or renaming.

## 1. Logo  →  `logo/`
Best to worst, give me whatever you can get:
1. **Vector** — `.svg`, `.ai`, `.eps`, or `.pdf` (scales perfectly, ideal for the header + favicon)
2. **High-res PNG** with a transparent background (the bigger the better)
3. Any logo file at all — I can work from it

> Note: the version on the live site is a downscaled web copy. If you can pull the
> original from Duda's **Media Manager** (or a brand/ADC file), that's the gold copy.
